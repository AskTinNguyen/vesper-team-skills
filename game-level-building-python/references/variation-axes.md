# Variation Axes

Use these axes to turn one generator into a reusable family.

## High-Value Variation Knobs

- footprint ratio: square, deep rectangle, wide rectangle
- vertical count: floors, tiers, tower caps
- roof family: flat, gabled, layered, pagoda, gatehouse
- symmetry mode: strict, near-symmetry, broken symmetry
- wing logic: none, left-right pair, rear wings, ring of side halls
- courtyard logic: none, shallow court, deep court, nested courts
- approach type: frontal stair, bridge, side ramp, rear stair
- corner treatment: plain, towers, buttresses, watch posts
- facade rhythm: sparse, even bay spacing, dense columns
- mesh palette: all primitive, mixed placeholder, project mesh override

## Recursive Library Growth

Start with:

- one base archetype
- three presets
- one compound composition pattern

Then grow in layers:

1. Base generator
2. Preset variants
3. Compound layout helper
4. District or landmark family

Each new script should either:

- add a new reusable module
- add a new preset family
- or compose existing generators into a larger layout

Avoid making a new standalone script if a preset or composition helper would solve the request.
