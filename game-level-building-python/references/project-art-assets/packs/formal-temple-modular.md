# Formal Temple Modular

Back to [Pack Guides](index.md) or [Project Art Assets](../index.md)

## Families

- `Asian_Modular_Temple`
- `USCANS_WindTemple`

## Best For

- clean modular walls
- main halls
- larger temple precinct shells
- more regularized architectural language

## How To Use It

- prefer `Asian_Modular_Temple` for wall and hall replacements in court-scale work
- reserve `USCANS_WindTemple` for larger precinct, fence, and shell work

## Caveat

This lane is strongest for formal architecture, not soft garden dressing.
