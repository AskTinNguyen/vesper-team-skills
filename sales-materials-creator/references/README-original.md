These markdown files are optimized for **[Gamma](https://gamma.app)** — import them directly to create beautiful, professional presentations.

### How to Use:

1. **Copy** content from `01-sales-deck.md` or `02-pitch-deck.md`

3. **Paste** the markdown content
4. **Choose** a theme (we recommend "Professional" or "Minimal" for B2B)

## 🎓 Training Exercises

### Exercise 1: Feeling Translation
Take any feature description and rewrite it using only feeling words.

**Example:**
- Feature: "54+ skills in the library"
- Feeling: "The confidence of having 54+ battle-tested playbooks at your fingertips"

### Exercise 2: The Mirror
Practice reflecting emotional states back to prospects.

**Example:**
> "So what I'm hearing is [pain]. And what you really want is [desire]. Does that sound right?"

### Exercise 3: Demo Feeling Check
After every demo practice, ask: "What feeling did that create?"
