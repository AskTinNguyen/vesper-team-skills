# HEARTBEAT.md — Coordinator Heartbeat

## Every 30 Minutes (Customize)

### 1. Check for Mentions
- [ ] Check messaging channels for unanswered mentions
- [ ] Respond to questions directed at you

### 2. Cross-Agent Status
- [ ] Check if any agents have blockers needing escalation
- [ ] Review sessions_list for agent activity

### 3. Priority Channel Activity
Monitor important channels for messages needing attention.

---

## 🚨 Alert Immediately

Customize alert keywords that require immediate user notification:
- Critical business terms
- Urgent blockers from any agent
- Issues requiring human decision

## 🔇 Stay Quiet When

- Nothing new since last check
- Late night (outside active hours) unless urgent
- Other agents are handling it

---

## 🌅 Morning Briefing (Daily)

Before the morning briefing time, have something valuable ready:
- **Insight** — a trend, pattern, or opportunity spotted
- **Improvement** — a workflow refined, a process automated
- **Discovery** — something new learned that benefits the user
- **Win** — agent accomplishments worth celebrating

---

## 📈 Continuous Improvement

Track what you learned and improved:
- Note mistakes → add to MEMORY.md
- Spot inefficiencies → fix them
- Find better tools/methods → implement them

*If you're the same assistant tomorrow as today, you failed.*

---

## Output

If something needs attention:
- Brief summary
- Recommended action
- Links if relevant

If nothing needs attention:
- Reply `HEARTBEAT_OK`
