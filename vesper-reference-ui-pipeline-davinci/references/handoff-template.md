# Handoff Template

Use this template when moving through the pipeline.

```text
Audience: <target user>
Primary action: <single core action>
Mandatory sections: <section 1>, <section 2>, <section 3>

Reference examples:
- <example 1>
- <example 2>
- <example 3>

Borrow from each:
- <structure>
- <interaction>
- <density / component treatment>

Avoid from each:
- <demo weakness>
- <copy weakness>
- <styling weakness>

Vesper upgrades:
- <hierarchy>
- <surface treatment>
- <copy>
- <interaction / disclosure>
- <contrast / color>

Implementation approach:
- <layout type>
- <component choices>
- <responsive behavior>
- <accessibility / semantics>

Polish checks:
- <states>
- <spacing>
- <copy>
- <motion restraint>
- <final product fit>
```

## Rule
Fill this once per meaningful screen, section, or component group. If the work spans many unrelated surfaces, split it into multiple handoffs.

If the final handoff still reads like a source remix instead of a Vesper-native interface, the translation step is incomplete.
